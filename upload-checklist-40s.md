# 40대 영양제 업로드 점검

## 업로드 전 확인

- 최종 영상 파일 존재 여부 확인
- 제목 문구가 과도하게 단정적이지 않은지 확인
- 설명문에 건강 정보 고지와 개인차 안내 포함 여부 확인
- 대표 썸네일 이미지 확인
- 업로드 후 유튜브 자동 자막 검수 항목 정리

## 현재 상태

- 대본 완료
- 장면 구성표 완료
- 대표 장면 이미지: 'output/40s' 하위 생성 완료
- TTS 텍스트: 'tts/40s' 하위 생성 완료
- TTS 오디오: 'tts_audio/40s' 하위 생성 완료
- 최종 영상: 'video_script/assemble-video.ps1'의 조립 계획 확인 완료
- 최종 mp4 조립: 'ffmpeg'와 'ffprobe' 미설치로 보류

## 자동 자막 검수 포인트

- '오메가3', '비타민D', '마그네슘' 표기 정확성
- 'EPA', 'DHA' 같은 영문 용어 인식 여부
- 수치, 복용량, 주의 문구 누락 여부

# 장면 구성표 01

## 프로젝트

- 주제: 30대라면 반드시 먹어야 하는 영양제 3가지
- 형식: 이미지+TTS 기반 페이스리스 영상
- 목표 길이: 약 10분
- 기준 대본: `script/draft-script-01.md`

## 제작 원칙

- 장면은 5초에서 20초 사이로 구성합니다.
- 무료 모델 이미지를 사용할 때는 이미지 안에 한글 텍스트를 넣지 않습니다.
- 무료 모델용 이미지 프롬프트는 영어 중심으로 작성합니다.
- 자막은 편집 단계에서 별도로 넣고, 이미지 자체에는 자막을 넣지 않습니다.

## 후속 작업

1. 각 장면별 자막 문장을 짧게 끊어서 별도 정리합니다.
2. 각 장면별 TTS 분량을 1문장 또는 2문장 단위로 분리합니다.
3. 이미지가 반복되는 장면은 프롬프트 변형본을 2~3개씩 추가합니다.
4. 편집 단계에서 자막, BGM, 전환 규칙을 붙입니다.

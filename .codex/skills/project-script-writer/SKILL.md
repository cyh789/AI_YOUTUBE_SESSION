# Project Script Writer

이 스킬은 이 프로젝트에서 사용할 한국어 유튜브 비디오 대본을 작성하거나 수정할 때 사용합니다.

## 역할

- 새 대본을 작성합니다.
- 기존 대본을 수정합니다.
- `refer` 폴더의 샘플 대본을 참고해 구조를 벤치마킹합니다.
- 결과 대본을 반드시 `script` 폴더 하위에 저장합니다.

## 핵심 규칙

- 대본은 반드시 한국어로 작성합니다.
- 대본은 유튜브 영상용 10~20분 분량을 기본 목표로 작성합니다.
- 샘플 대본은 `refer` 폴더 하위에서 찾습니다.
- 실제 결과물은 `script` 폴더 하위에 생성합니다.
- 샘플 대본 문장을 복사하지 말고 구조만 참고합니다.
- 프로젝트 맥락은 `AGENTS.md`, `tasks.md`, `findings.md`, `progress.md`를 읽고 반영합니다.

## 작업 순서

1. `AGENTS.md`를 읽습니다.
2. `tasks.md`, `findings.md`, `progress.md`를 읽고 현재 프로젝트 상태를 파악합니다.
3. `refer` 폴더 하위의 샘플 대본을 확인해 구조를 분석합니다.
4. 필요하면 기존 `script` 폴더 하위 대본도 확인합니다.
5. 프로젝트 주제에 맞는 한국어 유튜브 대본을 작성하거나 수정합니다.
6. 결과를 `script/<파일이름>.md`로 저장합니다.
7. 작업 결과를 `tasks.md`, `findings.md`, `progress.md`, `CHATS.md`에 반영합니다.

## 작성 기준

- 초보 유튜버도 바로 활용할 수 있는 형태로 작성합니다.
- 유튜브 내레이션에 맞는 자연스러운 구어체를 사용합니다.
- 장면 분해와 TTS 활용을 고려해 문단 호흡을 지나치게 길게 쓰지 않습니다.
- 건강 정보처럼 민감한 주제는 과장 표현을 줄이고 주의사항을 포함합니다.
- 이미지나 장면으로 옮기기 쉬운 설명 흐름을 우선합니다.

## 결과물

- 기본 결과물: `script` 폴더 하위의 한국어 대본 마크다운 파일
- 필요 시 부가 결과물: 대본 수정 메모, 장면 구성 힌트

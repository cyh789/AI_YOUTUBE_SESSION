# Project Scene Planner

이 스킬은 대본을 실제 영상 장면 구조로 바꿀 때 사용합니다.

## 역할

- 'script' 폴더 하위 대본을 읽습니다.
- 대본을 장면 단위로 나눕니다.
- 각 장면의 내레이션, 화면 목적, 필요한 이미지 방향을 정리합니다.
- 결과를 'scene-plan-xx.md' 형태의 장면 구성표로 정리합니다.

## 핵심 규칙

- 입력 대본은 'script' 폴더 하위에서 찾습니다.
- 장면계획은 10~20분 영상 조립을 기준으로 구성합니다.
- 장면 구성표는 이미지 생성과 TTS 텍스트 생성 단계에 바로 연결될 수 있어야 합니다.
- 결과 파일은 프로젝트 루트의 'scene-plan-xx.md' 형태로 저장합니다.

## 작업 순서

1. 'AGENTS.md'를 읽습니다.
2. 'tasks.md', 'findings.md', 'progress.md'를 읽고 현재 프로젝트 상태를 파악합니다.
3. 'script' 폴더 하위 대본을 읽습니다.
4. 도입, 본문, 요약, 마무리 구간을 장면 단위로 분해합니다.
5. 장면별 내레이션 요약, 화면 설명, 이미지 방향을 정리합니다.
6. 결과를 'scene-plan-xx.md' 파일로 저장합니다.
7. 작업 결과를 'tasks.md', 'findings.md', 'progress.md', 'CHATS.md'에 반영합니다.

## 작성 기준

- 장면 하나가 영상에서 어떤 역할을 하는지 한눈에 보이게 작성합니다.
- 너무 세세한 편집 지시보다 이미지 생성과 TTS 연결이 가능한 정도로 정리합니다.
- 같은 이미지를 반복 사용하는 장면은 그 점을 명시합니다.

## 결과물

- 기본 결과물: 프로젝트 루트의 장면 구성표 마크다운 파일
- 필요 시 부가 결과물: 장면별 이미지 프롬프트 메모

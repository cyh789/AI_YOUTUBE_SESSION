# Project Upload Checker

이 스킬은 유튜브 업로드 직전 점검할 항목을 정리할 때 사용합니다.

## 역할

- 최종 영상 파일 존재 여부를 확인합니다.
- 제목, 설명문, 썸네일, 자동 자막 검수 포인트를 점검합니다.
- 현재 프로젝트 기준으로 빠진 준비물이 없는지 체크합니다.
- 업로드 직전 체크리스트를 정리합니다.

## 핵심 규칙

- 최종 영상은 보통 'video_output' 폴더 하위에서 확인합니다.
- 썸네일 후보는 'output' 폴더 또는 별도 지정 파일에서 확인합니다.
- 자동 자막은 별도 자막 파일 제작보다 업로드 후 검수 포인트 중심으로 정리합니다.
- 점검 결과는 'findings.md' 또는 별도 체크리스트 마크다운으로 정리합니다.

## 작업 순서

1. 'AGENTS.md'를 읽습니다.
2. 'tasks.md', 'findings.md', 'progress.md'를 읽고 현재 프로젝트 상태를 파악합니다.
3. 최종 영상 파일 존재 여부를 확인합니다.
4. 제목 방향, 설명문 초안, 썸네일 후보, 자동 자막 검수 포인트를 점검합니다.
5. 빠진 항목과 수정 권장 사항을 체크리스트 형태로 정리합니다.
6. 결과를 관련 문서에 반영합니다.

## 작성 기준

- 업로드 전 바로 확인할 수 있는 짧은 체크리스트 중심으로 정리합니다.
- 자동 자막은 업로드 후 검수 포인트를 중심으로 작성합니다.
- 필수 항목과 권장 항목을 구분해도 좋습니다.

## 결과물

- 기본 결과물: 업로드 전 체크리스트 메모
- 필요 시 부가 결과물: 제목 후보, 설명문 초안, 썸네일 점검 메모
